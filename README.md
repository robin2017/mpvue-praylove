# mpvue-praylove

> 基于 mpvue 实现praylove微信小程序




## 页面

```bash
|__ test                                   # 测试
|__ match                                  # 匹配
    |__user-detail                         # 用户详情
|__ chat                                   # 聊天
    |__chat-detail                         # 聊天详情
|__ profile                                # 我的
    |__personal-info                       # 个人信息
    |__following                           # 我关注过谁
    |__follower                            # 谁关注过我
    |__about                               # 关于我们
```
